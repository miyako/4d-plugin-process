## Syntax

```
arguments:=PROCESS Get arguments
```

Parameter|Type|Description
------------|------------|----
arguments|TEXT|``json``

```
PROCESS SET VARIABLES (dict)
dict:=PROCESS Get variables
```

Parameter|Type|Description
------------|------------|----
dict|TEXT|``json``

```
list:=PROCESS Get list
```

Parameter|Type|Description
------------|------------|----
list|TEXT|``json``

```
id:=PROCESS Get id 
```

Parameter|Type|Description
------------|------------|----
id|LONGINT|
